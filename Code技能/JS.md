### Ajax

XMLHttpRequest()

有五个状态

| 状态 | 名称          | 描述                                           |
| ---- | ------------- | ---------------------------------------------- |
| 0    | Uninitialized | 初始化状态,XMLHttprequest已创建或者被abort重置 |
| 1    | Open          | open方法已调用,但是send方法未调用              |
| 2    | Sent          | send方法已调用, 但未收到response               |
| 3    | Receiving     | 响应头收到,响应体开始接收但未完成              |
| 4    | Loaded        | Http响应已完全接收                             |
